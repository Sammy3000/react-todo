# 📖 React todo App <a name="about-project"></a>

-Todo App created using react in order to consolidate my understanding on react basics. Can allow user to add, delete and edit tasks. Data is stored in local storage.

## 🛠 Built With <a name="built-with"></a>

- REACT js
- HTML
- CSS

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

1. Clone the Repo or Download the Zip file or `https://github.com/Sammy3000/React-todo-list.git`
2. cd todo-list
3. Open it with the live server

To run tests, run the following command:

After cloning the project you need to run these commands

`npm install` `This command will download all the dependancies of the project`

For tracking linter errors in CSS or SASS files run:

`npx stylelint "**/*.{css,scss}`

And For tracking linter errors in REACT files run:

`npx eslint "**/*.{js,jsx}"`
